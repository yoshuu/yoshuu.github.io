問題＿遇到錯誤 Error Cannot find module 'request'

發表於2022-08-10|更新於2023-05-17|JavaScript
|閱讀量:

在練習api時候遇到的

照著範例貼的也會報錯，無奈

後來解決方式是照著這一篇：
https://stackoverflow.com/questions/16482600/node-js-cannot-find-module-request

使用npm install request --save

問題就解決了
